# Estimation template

Use this when asking AI to estimate implementation work.

## Input

- Feature:
- Background:
- Known constraints:
- Affected systems:
- Deadline:

## AI prompt

```text
Read AGENTS.md and docs/project-context.md.
Estimate the work for the feature below.
Break it down into tasks, risks, unknowns, and validation work.
Do not implement anything.
```

## Output format

```md
# Estimate: <feature>

## Summary

## Assumptions

## Tasks

## Risks

## Unknowns

## Suggested implementation order

## Validation plan
```
